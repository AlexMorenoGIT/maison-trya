# Design — Site Web Maison trya.

## Stack
Next.js 15 + TypeScript + Tailwind CSS (site vitrine statique)

## Charte graphique

### Couleurs
- Tortoise Shell (marron foncé) : `#3C2413`
- Cloud Cream : `#FFF8EE`
- Blanc cassé : `#FBF9F8`
- Black : `#2B2B28`
- Rubber : `#754926`
- Gold : `#D3AC58`
- Gold foncé : `#636225`

### Logo
- Logo Cloud Cream `#FFF7EB` sur fond Tortoise Shell `#3C2413`
- Logo Tortoise Shell `#3C2413` sur fond Cloud Cream `#FFF8EE`
- Logo Cloud Cream `#FFF7EB` sur fond image
- Fichiers SVG officiels uniquement (jamais recréés)
- Zones d'exclusion respectées

### Typographie
- Police exclusive : Century Gothic (Regular + Bold)
- Titres en MAJUSCULES
- Alternance Bold (titres) / Regular (corps)

## Pages

### 1. Homepage
- **Header** : Logo bloc SVG centré, navigation centrée en dessous (COLLECTIONS, L'UNIVERS, À PROPOS), icônes recherche/panier à droite. Transparent sur hero, Cloud Cream au scroll.
- **Hero vidéo** : 100vh, vidéo placeholder (fond Tortoise Shell avec grain). Logo Cloud Cream superposé. Texte Century Gothic Bold : "ENTRE FÉROCITÉ ET VULNÉRABILITÉ". CTA : "DÉCOUVRIR LA COLLECTION".
- **Section manifeste** : Texte court + grande image latérale. Fond Cloud Cream.
- **Sélection 6 produits** : Grille 2 colonnes asymétriques. Nom (Bold, majuscules), prix (Regular), pastilles coloris.
- **Section éditorial** : Image plein écran + citation overlay + logo Cloud Cream.
- **Footer** : Fond Tortoise Shell, logo Cloud Cream, liens, newsletter.

### 2. Page Détail Produit
- Carrousel images (gauche), infos produit (droite)
- Nom (Bold, majuscules), prix, sélecteur coloris (pastilles), description, bouton CTA
- Section "VOUS AIMEREZ AUSSI" (3-4 produits)

### 3. Page À Propos
- Hero image plein écran + titre overlay Cloud Cream
- Section Vision (texte manifeste)
- Section 4 Valeurs (L'Instinct, L'Éveil, La Dualité, La Souveraineté) en grille avec images
- Section fondatrice (placeholder)

## Ambiance
Bohème luxe, grands espaces Cloud Cream, images plein écran, typographie aérée Century Gothic majuscules, tons chauds (marron, or, crème), éditorial haute couture.

## Marque
- Nom : Maison trya. (t minuscule, point final)
- Big Idea : "Entre férocité et vulnérabilité"
- Valeurs : L'Instinct, L'Éveil, La Dualité, La Souveraineté
